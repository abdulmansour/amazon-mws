# mws

Manipulate amazon seller data programatically

### Prerequisites<br/>
export your API credentials as environment variables in your shell:<br/>
```
export MWS_ACCOUNT_ID=...<br/>
export MWS_ACCESS_KEY=...<br/>
export MWS_SECRET_KEY=...<br/>
```
